# مشروع رسم مثلث باستخدام OpenGL

### وصف المشروع  
مشروع بلغة C++ باستخدام مكتبة OpenGL، يتيح للمستخدم إدخال إحداثيات النقاط الثلاث، ومن ثم يقوم البرنامج برسم مثلث بناءً على تلك الإحداثيات.

### تاريخ البدء  
17 أكتوبر 2022


### طريقة الاستخدام  
1. عند تشغيل البرنامج، يُطلب من المستخدم إدخال إحداثيات النقاط الثلاث (X, Y).  
2. بعد الإدخال، يتم رسم مثلث على الشاشة باستخدام مكتبة OpenGL.

ملف الرئيسي: triangle.cpp
الصورة المرفقة: output.png


### التقنيات المستخدمة  
- لغة C++  
- مكتبة OpenGL  
- برنامج Code::Blocks

### الهدف من المشروع  
تعزيز مهاراتي في برمجة الرسومات وتعلم كيفية التفاعل مع المستخدم ورسم الأشكال الهندسية باستخدام OpenGL.

### ملاحظات  
هذا المشروع جزء من مجموعة مشاريع تدريبية أعمل عليها باستمرار لتطوير نفسي في مجال البرمجة.
